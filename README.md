# ML Projects for Beginners

This repository contains **5 beginner-friendly machine learning projects** built as Google Colab notebooks. Each project uses a real Kaggle dataset, includes step-by-step explanations, and runs entirely in the browser — no local Python installation required. Whether you're a student taking your first ML course or a developer curious about data science, these projects are designed to get you from zero to a working model in under an hour.

## How to Run on Google Colab

1. Go to [https://colab.research.google.com](https://colab.research.google.com)
2. Click **File → Upload Notebook** and upload the `.ipynb` file for the project you want to run
3. Go to the Kaggle dataset URL listed inside the notebook, create a free Kaggle account if you do not have one, and download the CSV file to your computer
4. Run the first code cell in the notebook — it will show a file upload button; click it and upload the CSV you downloaded from Kaggle
5. Click **Runtime → Run All** to run all cells from top to bottom
6. Read the comments in each cell as you scroll through to understand what is happening
